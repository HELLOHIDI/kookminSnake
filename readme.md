### 빌드방법

```
// 의존 라이브러리 설치 필요
sudo apt-get install -y libncurses5-dev libncursesw5-dev
// clone 또는 tarball 이용
git clone https://github.com/Simhyeon/kookminSnake
cd kookminSnake
make
```
